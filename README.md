# 📊 Sales Performance Dashboard

## 🧩 Objective
To visualize regional sales performance and key KPIs using Power BI.

## 📊 Tools Used
- Power BI for dashboard design
- Excel for data preprocessing

## 💡 Key Insights
- Region East had the highest sales, but Region West had the highest profit margin.
- Product Line B underperformed consistently over 3 quarters.
